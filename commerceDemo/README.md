## 프로젝트 주제
- 쇼핑몰 DEMO 버전 제작   
   
## 프로젝트 목표
1. 독립형 쇼핑몰을 제작해보며 해당 도메인에 대한 이해력을 높인다.
2. Mockito를 사용하여 독립된 테스트를 진행한다.
3. JPA를 이용하여 SQL코드의 반복을 줄이며 객체지향적으로 데이터를 관리한다.

## 사용기술
- Springboot
- Spring security
- Spring data jpa
- Thymeleaf
- H2(Database)
- Maven
